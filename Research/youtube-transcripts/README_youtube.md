# YouTube Transcripts

This folder contains transcript-based summaries of long-form videos, podcast appearances, and outbound sales interviews from selected B2B SaaS cold outreach practitioners.

Files are organized by expert.

Each file includes:

* Video title
* URL
* Publication date
* Transcript summary
* Key insights

The goal is to identify recurring patterns across prospecting, messaging, deliverability, infrastructure, AI, and outbound execution.

These transcript summaries support the synthesized findings documented in `Research/insights.md`.
